## WordPress Integration

### Importing Templates to Elementor

1. In WordPress, go to **Elementor > Tools**
2. Click the **Import** tab
3. Upload the JSON file you want to import
4. Go to **Pages > Add New** or edit an existing page
5. Click **Edit with Elementor**
6. Click the folder icon (templates) > **My Templates**
7. Insert the imported template
